# tasks-frontend

Aplicação web para gerenciar tarefas. [Ver API](https://github.com/tiagoon/tasks-api)


## Configuração do Projeto

### Crie o .env e aponte para a sua API

```
cp .env.example .env
```


### Instale as dependências

```
yarn install
```

### Inicie o servidor local

```
yarn serve
```

### Crie o build
```
yarn build
```

## Definições

Usando `Nodejs v14`
